# JAVASCRIPT DEEP DIVE STUDY SPACE

이 Repository는 **JavaScript Deep Dive** 책을 2회독하면서 면접에 나올만한 질문들을 서로 공유하는 공간입니다.

총 49장으로 구성되어 있으며, 하루에 2장씩 읽고, 필요한 내용들을 정리하고, 면접에 나올만한 질문을 스스로 생각해보며, 이해되지 않는 내용들을 정리하여 남겨봅니다.
